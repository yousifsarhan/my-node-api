Node.js REST API — Portfolio Backend

This REST API powers my personal portfolio projects and demonstrates backend development using Node.js and Express. It exposes structured JSON endpoints following REST principles.

API Endpoints

Method Endpoint Description
GET / API documentation page
GET /api/projects Retrieve all projects
GET /api/projects/:id Retrieve a project by ID
POST /api/projects Create a new project
DELETE /api/projects/:id Delete a project
